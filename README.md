# 🚀 DEv-War

DevWar is a competitive learning tracker designed to turn daily progress into a measurable war.

Track your hours.  
Earn XP.  
Build streaks.  
Challenge others.  
Level up.

---

## 🔥 Why DevWar?

Most people start learning with motivation.  
Very few stay consistent.

DevWar gamifies learning by combining:
- Progress tracking
- XP & leveling system
- Streak mechanics
- Peer competition

The goal is simple:
Make discipline addictive.

---

## 🛠 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Supabase (Authentication + Database)

Architecture:
Frontend directly communicates with Supabase using secure RLS policies.

---

## ✨ Features

- User authentication (signup / login)
- Daily learning hour logging
- XP and level calculation
- Streak tracking
- Project and task management
- War mode (user vs user comparison)
- Responsive dark-themed UI

## 🔐 Security

- Supabase Row Level Security (RLS) enabled
- Users can only access and modify their own data
- No service role keys exposed
